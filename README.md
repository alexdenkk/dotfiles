# dotfiles
 My dotfiles using alpine/i3/i3status/kitty/dunst/micro

# usage
 1. Clone repository
 ```sh

 git clone https://github.com/alexdenkk/dotfiles
 cd dotfiles
 ```
2. run installation script
```sh
sh install
```
